LESSON 1: Introduction

Web Basics

The majority of the web is made up of HTML connected together via hyperlinks

The main components of the Web consist of :
- Me
- Internet
- Computers
- Servers

Servers are computers that host files and uses HTTP for communication between the internet and other computers.

HTML Overview

HTML files contain elements which make up the content of the document and they are surround by tags. Tags can be used to modify the formatting of the element for example using bold or italic text.

HTML attributes are tools to add links and images into the document. href are used to add a URL and src are used for images. More can be found at this link.

HTML always follow a basic structure, starting with !DOCTYPE indicating it is an HTML file, followed by a head usually containing a title and then followed by a body with the bulk of the content.

LESSON 3: More CS

More CSS

CSS - Breakdown

When writing CSS it is broken down into 4 main components. Starts with the selector which is used to target the elements that are assigned to a class or attribute. Within the selector there is a declaration which is divided into the property and value of that property.
When applying a selector it will apply the same declarations to all the similar tags within the HTML document. Another important part is to apply prefixes such as webkit,moz,ms so that the CSS style will work across different browsers.

The Box Model

HTML documents are always organised into boxes and using CSS we can manipulate to obtain our desired outcome. Boxes have many properties such as the margin, border padding and content. Using a box-sizing declaration can help keep the box the same size even if the page changes.
Boxes can be changed depending to pixi number or as a percentage to the page, it is important to set the appropriate parameters depending on the wanted outcome of the box.
