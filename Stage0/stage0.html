Web Basics

The majority of the web is made up of HTML connected together via hyperlinks

The main components of the Web consist of :
- Me
- Internet
- The internet
- Servers

Servers are computers that host files and uses HTTP for communication between the internet and other computers.

HTML Overview

HTML files contain elements which make up the content of the document and they are surround by tags. Tags can be used to modify the formatting of the element for example using bold or italic text.

HTML attributes are tools to add links and images into the document. href are used to add a URL and src are used for images. More can be found at this link.

HTML always follow a basic structure, starting with !DOCTYPE indicating it is an HTML file, followed by a head usually containing a title and then followed by a body with the bulk of the content.
